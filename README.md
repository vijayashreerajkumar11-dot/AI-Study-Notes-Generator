# 📚 AI Study Notes Generator

An interactive AI-inspired Study Notes Generator built using **Python** and **Streamlit**. The application allows users to enter a topic and instantly receive concise study notes through a simple and user-friendly interface.

---

## 🚀 Features

- 📖 Generate study notes for selected topics
- 💻 Simple and interactive Streamlit interface
- 📝 Stores searched topics in a history file
- ⚡ Fast and easy to use
- 🎯 Beginner-friendly project

---

## 🛠️ Technologies Used

- Python
- Streamlit

---

## 📂 Project Structure

```
AI-Study-Notes-Generator/
│
├── app.py              # Streamlit user interface
├── notes.py            # Generates notes for topics
├── requirements.txt    # Project dependencies
├── history.txt         # Stores searched topics
└── README.md
```

---

## ▶️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/vijayashreerajkumar11-dot/AI-Study-Notes-Generator.git
```

### 2. Navigate to the project folder

```bash
cd AI-Study-Notes-Generator
```

### 3. Install the required package

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```

## 💡 Available Topics

Currently, the application provides detailed notes for:

- Python
- Machine Learning
- Artificial Intelligence

For other topics, it displays a basic introduction.

## 📸 Output

Enter a topic and click **Generate Notes** to view the generated study notes.
